
This package contains:
- Training plans for Module 1 and Module 2 (see `Module 1/` and `Module_2/`)
- A repository of Python and Java exercises (`training_repo/`)
- Example prompts, quizzes, and reflection templates
- Slide decks for each module (see `Slides/`)
